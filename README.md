## Este projeto tem como objetivo traduzir o software EST para Português (Brasil).

*O EST é um ambiente de programação visual para robótica educacional amplamente utilizado em kits educacionais baseados na plataforma Dr.Luck.*


**Este repositório NÃO possui vínculo com os desenvolvedores originais.**
-
Todos os direitos do software original pertencem aos seus respectivos autores.

Este projeto distribui:
- Arquivos de instalação base
- Patch de Tradução PT-BR
- Documentação técnica traduzida
